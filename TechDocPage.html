Conditional execution

Often, we want a program to behave differently based on some condition. Conditional statements allow this.

One example is the if statement:

if x > 0 :
  print('x is positive')

The boolean expression that comes after if is the condition. The condition expression often uses comparison and logical operators. Here are some examples:

  • Comparison operators
    • x is equal to y: x == y
    • x is not equal to y: x != y
    • x is greater than y: x > y
    • x is less than y: x < y
  • Logical oeprators
    • x > 0 and x < 10
    • x > 0 or x < 10
    • not(x > 0)
Functions

A function is a named sequence of statements that performs a computation. Functions may require arguments, which are input values to the function for computation.

Python has a number of built-in functions. Common examples are type, max, and min:

  • >>> type(32)

  • >>> max('Hello world')
'w'
  • >>> min('Hello world')
' '

It is also possible to define new functions. A function definition specifies the name of a new function and the sequence of statements that execute when the function is called. After defining the function, it can be reused over and over throughout the program. For example:

def print_twice(bruce):
  print(bruce)
  print(bruce)

def is a keyword that indicates that this is a function definition. The name of the function is print_twice. The function takes one argument, bruce. The rules for function and argument names are the same as for variable names: letters, numbers and some punctuation marks are legal, but the first character canâ€™t be a number. Avoid keywords. Functions, arguments, and variables should all have distinct names.

Iteration

Iteration is useful for performing repetitive tasks. Python has several features that allow this:

  • the while statement
  • the for loop

Here is an example of a while statement:

n = 5
while n > 0:
  print(n)
  n = n - 1
print('Blastoff!')

The while statement runs in the following manner:

  1. Evaluate the condition, yielding True or False.
  2. If the condition is false, exit the while statement and continue execution at the next statement.
  3. If the condition is true, execute the body and then go back to step 1.

Here is an example of a for loop:

friends = ['Joseph', 'Glenn', 'Sally']
for friend in friends:
  print('Happy New Year:', friend)
print('Done!')

The friends variable is a list of three strings. The for loop goes through the list and executes the body once for each of the three strings in the list, resulting in this output: 

Happy New Year: Joseph
Happy New Year: Glenn
Happy New Year: Sally
Done!
Lists

A list is a sequence of values. Lists are mutable, meaning the order of items can be changed, or a list item can be reassigned. A list element is accessed or reassigned by specifying the index in brackets. For example:

>>> numbers = [17, 123]
>>> numbers[1] = 5
>>> print(numbers)
[17, 5]

A for loop is commonly used to go through a list.

Lists can be concatenated with a + operator, or repeated with a * operator.

Lists can be sliced in the following manner:

>>> t = ['a', 'b', 'c', 'd', 'e', 'f']
>>> t[1:3]
['b', 'c']
Dictionaries

A dictionary is similar to a list, except that the index can be any unique value instead of running integers in the case of a list. These indices in a dictionary are called keys. Here is an example:

>>> eng2sp = {'one': 'uno', 'two': 'dos', 'three': 'tres'}
>>> print(eng2sp)
{'one': 'uno', 'three': 'tres', 'two': 'dos'}

Like lists, a for loop is commonly used to go through a dictionary. The difference here is that the for loop traverses each key in the dictionary. Here is an example:

counts = { 'chuck' : 1 , 'annie' : 42, 'jan': 100}
for key in counts:
  print(key, counts[key])
